===========================
Guide d'Utilisation
===========================

Bienvenue dans **Orange County Lettings**, une plateforme pour explorer les profils d'utilisateurs et les annonces de locations de vacances. Ce guide explique les fonctionnalités principales auxquelles vous avez accès en tant qu'utilisateur.

Introduction
============

L'interface utilisateur est simple à naviguer avec deux principales sections :
- **Profiles** : Liste des profils utilisateurs disponibles.
- **Lettings** : Liste des annonces de location disponibles.

Les utilisateurs peuvent consulter les détails des profils et des annonces en quelques clics.

Naviguer dans l'application
===========================

Page d'Accueil
--------------

Lorsque vous accédez à l'application, vous arrivez sur la **page d'accueil**. Elle vous accueille avec un message chaleureux et propose deux actions principales via des boutons :

- **Profiles** : Accédez à la liste des profils utilisateurs.
- **Lettings** : Accédez à la liste des annonces de location disponibles.

Vous pouvez cliquer sur l'un des boutons pour explorer l'une des sections de l'application.

Liste des Profils
-----------------

En cliquant sur le bouton **Profiles** ou en naviguant directement à l'URL `/profiles/`, vous arriverez sur la page de **liste des profils**. Cette page affiche une liste de tous les utilisateurs inscrits avec un profil public. Chaque profil est cliquable et vous pouvez consulter les détails en cliquant sur le nom d'utilisateur.

Si aucun profil n'est disponible, un message indiquant "No profiles are available" sera affiché.

Détails d'un Profil
-------------------

Lorsque vous sélectionnez un utilisateur, vous serez redirigé vers une page avec des informations détaillées sur le profil sélectionné, incluant :
- Le **prénom** et **nom de famille**.
- L'**email** de l'utilisateur.
- La **ville préférée** de l'utilisateur.

Vous avez également la possibilité de retourner à la liste des profils, aux annonces ou à la page d'accueil à l'aide des boutons fournis.

Liste des Locations
-------------------

En cliquant sur le bouton **Lettings** ou en naviguant à l'URL `/lettings/`, vous serez redirigé vers la **liste des annonces de location**. La page affiche les annonces de location disponibles. Chaque annonce est cliquable et vous redirige vers une page dédiée aux détails de cette annonce.

Si aucune annonce n'est disponible, un message indiquant "No lettings are available" sera affiché.

Détails d'une Annonce
---------------------

Lorsque vous sélectionnez une annonce de location, vous accédez à une page détaillant l'annonce. Les informations incluent :
- L'**adresse complète** : numéro, rue, ville, état, code postal, et code pays ISO.

Sur cette page, vous pouvez retourner à la liste des annonces, aux profils ou à la page d'accueil via les boutons de navigation.

Navigation Globale
==================

À tout moment, vous pouvez utiliser la barre de navigation située en haut de chaque page pour accéder rapidement à :
- **Home** : Retourner à la page d'accueil.
- **Profiles** : Voir la liste des profils.
- **Lettings** : Voir la liste des annonces de location.

Footer
------

En bas de chaque page, vous trouverez un **footer** avec des liens supplémentaires pour consulter les politiques de confidentialité et les conditions d'utilisation du site.

==========================
